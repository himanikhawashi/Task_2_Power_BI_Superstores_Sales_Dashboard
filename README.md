# Superstore Sales Dashboard using Power BI

## Project Overview
This project analyzes Superstore sales data using Microsoft Power BI. It includes interactive dashboards and a 15-day sales forecast to support business decision-making.

## Tools Used
- Microsoft Power BI
- Superstore Sales Dataset

## Dashboard Features
- Total Sales
- Total Profit
- Total Quantity
- Average Delivery Time
- Sales Trend
- Profit Trend
- Sales by Category
- Sales by Sub-Category
- Sales by Region
- Sales by State
- Sales by Payment Mode
- Sales by Ship Mode
- 15-Day Sales Forecast
